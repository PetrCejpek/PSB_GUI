Groups separation

Group separator tool (see the figure below) serves, as the name suggests, for a separation of datapoints into groups. The set of points currently included in the selected group is represented by the ticks in the column In group inside the Data table. The default datapoint group is set as all peaks and no point can be unticked.

To add a new group just click on the New button. Then select the newly created group (by clicking on it in Group separator section) and you will be allowed to tick or untick the tickboxes inside the Data table. The point choice is not group exclusive - if needed, the point can belong to several number of groups. The selected/active group for which you are currently changing the parameters is highlighted in the yellow box under the data table. In the example of abovementioned figure, the user is currently working with the default Group #0: All peaks.

The group separator can be also used to exclude apparently wrong datapoints. For example, the peaks had the low intensity, so the corresponding fits during the preprocessing were bad and have produced refined parameters with the strange or non-physical values. In such a case just create a new group and untick datapoints which you want to exclude from the further refinement.

If you want to remove a group, select it by clicking on it in the Group separator section and press the Remove button.

IMPORTANT: If you want to perform a refinement of the datapoints corresponding to the individual groups, you need to tick the Fit groups tickbox which is placed on the right of the Fit button. Otherwise, the refinement will be still performed on the all peaks (default Group #0).

Quick selection tool

If the labels are available in the imported data, Quick selector could be used for the fast ticking of datapoitns. Naturaly, this selector is not available for the All peaks group.

After launching the Quick selector, user can choose the condition for the ticking based on the labels introduced during data preprocessing. Quick selection tool will find unique labels in the dataset. Then, the user can select the labels containing or being exactly the specific string.

Introduction

PSB_GUI is the Matlab form application developed for the refinement of the X-ray diffraction (XRD) line shift and broadening to obtain information about the residual stress, crystallites sizes and microstrain in cubic materials. The application has been created with the guide enviroment. To run it properly, Matlab Optimalisation toolbox is required.

Before own usage of this program, the experimental data input should be carefully prepared by a user. This requires a refinement of the measured data (for example 2θ-scans of individual Bragg reflections) by an external function or script. The resulting input data should be provided in a tabulated format in a text file or a spread sheet. See also Data import section.

The main aspect of PSB_GUI is the separation of individual experimental points into groups. Each group can be then connected to its own combination of theoretical models for the following refinement. The only condition is to have the experimental data corresponding only to one sample measured at the same conditions.

Main window

The main window of PSB_GUI can be separated into several parts (see the figure):

In the Input data section, you can import your preprocessed experimental data and/or manually specified them if the semi-automatic input didn't work as expected. The Group separator section serves for the separation of your datapoints into groups, or to exclude obviously wrong datapoints from the fit. In the Material information part, you can specify the available information of your studied material. The Fit and models options allow you to choose various theoretical models and fitting parameters for your refinement. Each of your specified groups of datapoints has its own models and is fitted separately*. The results of the last successful refinement are plotted in the figures on the left side of the main window and also printed into Results section together with the own Matlab console.
